# Fun Game Patterns

These notes guided the second pass of Number Tug.

## Patterns Used

- Speed matters: Kahoot-style scoring rewards players for answering quickly, so Number Tug adds a speed bonus to each correct pull.
- Player identity matters: Blooket lets students enter nicknames and choose a character before play, so Number Tug now has player names and face choices.
- Rewards matter: Prodigy wraps math practice in visible rewards and progress moments, so Number Tug adds streak bonuses, math bursts, and more energetic feedback.
- Browser haptics are limited: the Vibration API is not supported in every major browser, so Number Tug uses best-effort vibration and keeps sound/visual feedback as the reliable layer.
